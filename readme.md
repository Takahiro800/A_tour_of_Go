## 複数の変数展開

```go
fmt.Printf("%T,%v", z,z)
```

- `%T`で型表示
- `%z`で値表示

## 配列とスライスについて

- スライスは、可変長の配列。配列よりも一般的である。
宣言は `[]T`。長さを指定すると配列。指定しないとスライスの宣言となる
